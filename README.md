# Eedox

This an experimental and educational UNIX-like kernel I have been working on for a while!

## Kernel Features

- paging
- Multitasking : RR preemptive scheduling
- syscalls
    - exit
    - open
    - read
    - write
    - ...
- ATA disk controller
- a minimal file system
- creating and managing child processes
    - fork
    - wait
    - ...
- IPC
    - pipes
    - message queues

## Other Features:

- core utils
    - cp
    - mkdir
    - stat
    - wc
    - ...
- shell interpreter
- standard library
